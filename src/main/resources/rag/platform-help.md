# 网球场预约平台 — 用户帮助摘要

## 预约与时段

- 用户需登录后创建预约；预约需指定场地、日期、开始与结束时间（HH:mm）。
- 下单前系统会校验时段是否与已有预约冲突；最终以「可约校验」工具结果为准。
- 订单创建后默认进入「待付款」状态，请在规定时间内完成支付。

## 支付与渠道

- 支持的支付渠道包括：微信支付、支付宝、闲鱼等（以实际页面与订单说明为准）。
- 付款后订单进入已付款/待使用等状态，具体以订单详情显示为准。

## 取消与退款

- 取消预约需用户明确确认后，通过平台发起取消申请或按订单规则操作。
- 退款规则以平台公示及订单状态为准；如有疑问请联系人工客服或查阅最新公告。

## 优惠券与活动

- 优惠券、秒杀等活动可能有数量与时间限制；领取或使用规则以活动页说明为准。
- 下单时若支持优惠券抵扣，请在结算流程中按提示输入或选择券码。

## 联系方式

- 紧急问题请通过 App/站内渠道联系管理员；本摘要仅作智能客服辅助参考。
